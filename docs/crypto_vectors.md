# Crypto Vectors

This file is generated by `scripts/generate_vectors.go`.

Run the generator to populate test vectors:

```bash
go run scripts/generate_vectors.go --output docs/crypto_vectors.md
```

The generated output includes:

- derived signing key
- public key
- blinded message
- blind signature
- DLEQ proof values
- unblinded signature
- witness
- nullifier
